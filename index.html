About

I am a marketer turned software engineer with an interest in building tools to automate daily tasks. I am always looking for more opportunities to learn new skills, programming languages and frameworks. I am currently working on a personal recipe project using React Hooks and PostgreSQL with REST-ful API backend routes.

I have created a full-stack single page web application called Kineto (a Netflix clone). I leveraged a database using PostgreSQL with REST-ful API backend routes as well as React, Redux, and CSS. I also worked on a budgeting web application called Making Cent$ with a few teammates, leveraging the MERN (MongoDB, Express.js, React, Node.js) web development stack.
